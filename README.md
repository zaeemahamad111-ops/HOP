# HOP — House of Perfume

A cinematic, premium e-commerce experience for a niche Indian fragrance house. Built with Next.js, Framer Motion, and GSAP.

## ✨ Features
- **Cinematic Hero**: Scroller-based frame animation sequence.
- **Interactive Philosophy**: Responsive glowing roots and editorial pillars.
- **Scent Quiz**: Ritualistic discovery flow for signature blends.
- **Boutique Shop**: High-concentration fragrance archive with real-time cart.
- **Admin Dashboard**: Full product and order management system.
- **Responsive Design**: Mobile-first luxury experience.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP
- **State Management**: Zustand
- **Database**: Local JSON Archive (`db.json`)

## 🚀 Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Run the development server: `npm run dev`.
4. Open [http://localhost:3000](http://localhost:3000).

## 📦 Project Structure
- `src/components`: Interactive UI components.
- `src/app/shop`: The product archive.
- `src/app/admin`: The Shopify-style backend.
- `data/db.json`: Product and order data.
